Explicación

Este ejercicio muestra cómo se utilizan los operadores de igualdad débil (== y !==) en JavaScript.

El operador == compara solo el valor, mientras que el operador !== compara tanto el valor como el tipo de dato.

En este caso, se han utilizado estos operadores para mostrar cómo se pueden comparar valores y tipos de datos de manera diferente.
